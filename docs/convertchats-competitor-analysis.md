## 2. Features & Product Suite

ConvertChats has four core product pillars plus one consumer product:

### A. Broadcast / Outreach (WhatsApp + Telegram)

- 10,000 to 100,000+ messages per day capacity
- Sub-500ms delivery latency
- "Tunnel architecture" that avoids routes heavily taxing marketing messages
- Dynamic content variation to evade AI-based content filters
- Invisible character insertion for pattern avoidance
- Randomized sending intervals simulating human behavior
- Time-zone-aware delivery
- Daily volume caps per number
- Media support: images, video, PDFs, clickable CTA buttons
- Active-Only Validation (pre-scans numbers to verify WhatsApp registration)
- Automated failover rotation -- if a sending number gets blocked (403), traffic reroutes to pre-warmed standby numbers with zero downtime
- A/B testing on copy and timing
- No Meta API approval required -- uses existing phone numbers
- No per-message markup fees (vs. competitors' 15-20% markup)
- No template approval bottleneck

### B. CRM (Messaging-Native)

- Shared inbox consolidating WhatsApp + Telegram into one interface
- Role-based access controls
- Automated and manual lead routing/distribution with load balancing
- Real-time delivery, read, and reply tracking
- Conversation history with full context
- Pipeline management

### C. Monitoring (WhatsApp, Telegram, WeChat)

- Live Sentry real-time dashboard showing agent activity, response times, conversion rates
- Full conversation visibility for managers
- Sentiment Scan analysis (higher tiers)
- Conversational Health Score (higher tiers)
- CSAT tracking
- Agent performance metrics

### D. AI Agent ("Convex AI")

- 0.4-second average response time
- 24/7 availability across all time zones
- Two modes: Sales Mode and Support Mode
- Learns from knowledge bases, FAQs, and past conversations
- Autonomous lead qualification and CRM population
- Smart handoff to human agents
- Claims "3x more closed deals" in first quarter
